LAYOUT

El Layout es la forma en que se van a distribuir los distintos elementos en las distintas secciones de nuestra página web.

Etiquetas Layout

HTML nos ofrece varias etiquetas semánticas que definen diferentes partes de una web, tales como...

  • <header> Define el encabezado del documento o de una sección
  • <nav> Define un contenedor para enlaces de navegación
  • <section> Define una sección del documento
  • <article> Define un artículo
  • <aside> Define contenido posicionado a un costado
  • <footer> Define el pie de página del documento
  • <details> Define detalles adicionales
  • <summary> Define un encabezado para el <details>
  • <figure> - <figcaption> Una imagen con su título se pueden agrupar dentro de <figure>, poniendo el título en la etiqueta <figcaption>

Métodos Layout

Hay 5 formas de crear un sistema de Layout, cada una tiene sus ventajas y desventajas.

  • Tablas HTML (no recomendado)
  • CSS float
  • CSS framework
  • CSS Flexbox
  • CSS Grid

Tablas HTML

El elemento <table> no se creó para crear layouts. Así que no deberíamos usar este método para crear un Layout.

CSS Floats

Es una forma muy común de maquetar toda la web usando la propiedad de 'CSS Float'. Es fácil de aprender, pero en algunos casos puede que no sea tan flexible como necesitamos.

CSS Framework

Es la forma más rápida de crear un Layout. Un ejemplo de framework es Bootstrap

CSS Flexbox

Es un nuevo sistema de Layout disponible desde CSS3. Se va ajustando a las diferentes resoluciones de pantalla. No funciona con Internet Explorer 10 :'c

CSS Grid

CSS Grid nos ofrece un Layout basado en una rejilla con Columnas (Columns) y Filas (Rows). Es fácil e intuitivo posicionar los elementos. No funciona con Internet Explorer ni en Edge 15 o inferiores :'c



&nbsp;

Significa: Non Breakable Space, útil para representar múltiples espacios en blanco, y para hacer dos trozos de texto inseparables, es decir, por ejemplo: que no haya un salto de línea (Non Breakable Space) entre '10' y 'km/h'

EL ATRIBUTO CLASS!

El atributo 'class' de HTML se usa para definir estilos iguales para varios elementos al mismo tiempo. Es case sensitive.

En CSS se usa el '.' seguido del nombre de la 'class' para selecccionar los elementos que pertenezcan a la misma.

El atributo 'id' especifica un identificador único para un elemento HTML.

El valor del 'id' se usa en CSS y JavaScript para aplicar estilos o tareas sobre el elemento.

En CSS se usa el '#' para selecccionar los elementos por su 'id'
